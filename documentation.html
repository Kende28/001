API Dokumentáció - Energiaitalok

Ez az API dokumentáció az energiaitalok kezeléséhez szükséges végpontokat tartalmazza.

1. POST /energy-drinks

Leírás: Új energiaital hozzáadása az adatbázishoz.

Request Body:

{
  "name": "Red Bull",
  "brand": "Red Bull",
  "caffeine": 80,
  "sugar_free": false
}

Válasz:

{
  "message": "Energiaital sikeresen hozzáadva"
}

Hibák:

Státuszkód: 400 - Ha bármelyik kötelező mező hiányzik:

{
  "arrayError": [
    { "error": "A 'name' mező megadása kötelező." },
    { "error": "A 'brand' mező megadás kötelező." },
    { "error": "A 'caffeine' mező megadás kötelező." },
    { "error": "A 'sugar_free' mező megadás kötelező." }
  ]
}

Státuszkód: 500 - Ha szerverhiba történik a mentés során:

{
  "error": "Szerverhiba történt az energiaital mentés közben"
}

2. GET /energy-drinks

Leírás: Az összes energiaital lekérése.

Válasz:

[
  {
    "id": 1,
    "name": "Red Bull",
    "brand": "Red Bull",
    "caffeine": 80,
    "sugar_free": false,
    "created_at": "2023-10-01T10:00:00",
    "updated_at": "2023-10-01T10:00:00"
  },
  {
    "id": 2,
    "name": "Monster",
    "brand": "Monster",
    "caffeine": 90,
    "sugar_free": true,
    "created_at": "2023-10-02T10:00:00",
    "updated_at": "2023-10-02T10:00:00"
  }
]

Hibák:

Státuszkód: 500 - Ha szerverhiba történik a lekérés során:

{
  "error": "Szerverhiba történt az energiaitalok lekérdetése közben"
}

3. DELETE /energy-drinks/:id

Leírás: Energiaital törlése az adatbázisból.

URL paraméter: :id - A törlendő energiaital azonosítója

Válasz:

{
  "message": "Energiaital sikeresen törölve"
}

Hibák:

Státuszkód: 404 - Ha az energiaital nem található:

{
  "error": "Energiaital nem található"
}

Státuszkód: 500 - Ha szerverhiba történik a törlés során:

{
  "error": "Törlési hiba"
}

4. PUT /energy-drinks/:id

Leírás: Energiaital frissítése.

Request Body:

{
  "name": "Monster Energy",
  "brand": "Monster",
  "caffeine": 90,
  "sugar_free": true
}

Válasz:

{
  "message": "Energiaital sikeresen frissítve"
}

Hibák:

Státuszkód: 400 - Ha a kötelező mezők hiányoznak:

{
  "error": "Hiányzó vagy hibás mezők"
}

Státuszkód: 404 - Ha az energiaital nem található:

{
  "error": "Energiaital nem található"
}

Státuszkód: 500 - Ha szerverhiba történik a frissítés során:

{
  "error": "Szerverhiba történt az energiaital módosítás közben"
}
