API Dokumentáció - Energiaitalok

Ez a dokumentáció a energiaitalok API használatát ismerteti. Az API lehetővé teszi energiaitalok hozzáadását, lekérdezését, frissítését és törlését.

POST /energy-drinks

Leírás: Új energiaital hozzáadása az adatbázishoz.

Kérés:

POST /energy-drinks

Példa:

{
    "name": "Red Bull",
    "brand": "Red Bull GmbH",
    "caffeine": 80,
    "sugar_free": true
}

Válasz:

{
    "message": "Energiaital sikeresen hozzáadva"
}

Hibák:

400 - Hiányzó mezők: 'name', 'brand', 'caffeine', 'sugar_free'

500 - Szerverhiba az energiaital mentése közben

GET /energy-drinks

Leírás: Minden energiaital lekérdezése.

Kérés:

GET /energy-drinks

Példa válasz:

[
    {
        "id": 1,
        "name": "Red Bull",
        "brand": "Red Bull GmbH",
        "caffeine": 80,
        "sugar_free": true,
        "created_at": "2023-10-02T12:00:00",
        "updated_at": "2023-10-02T12:00:00"
    },
    {
        "id": 2,
        "name": "Monster",
        "brand": "Monster Energy",
        "caffeine": 160,
        "sugar_free": false,
        "created_at": "2023-10-02T12:30:00",
        "updated_at": "2023-10-02T12:30:00"
    }
]

Hibák:

500 - Szerverhiba az energiaitalok lekérdezése közben

DELETE /energy-drinks/:id

Leírás: Energiaital törlése az adatbázisból az azonosító alapján.

Kérés:

DELETE /energy-drinks/:id

Példa:

DELETE /energy-drinks/1

Válasz:

{
    "message": "Energiaital sikeresen törölve"
}

Hibák:

404 - Energiaital nem található

500 - Törlési hiba

PUT /energy-drinks/:id

Leírás: Energiaital frissítése az azonosító alapján.

Kérés:

PUT /energy-drinks/:id

Példa:

{
    "name": "Red Bull Sugarfree",
    "brand": "Red Bull GmbH",
    "caffeine": 80,
    "sugar_free": true
}

Válasz:

{
    "message": "Energiaital sikeresen frissítve"
}

Hibák:

400 - Hiányzó mezők: 'name', 'brand', 'caffeine'

500 - Szerverhiba az energiaital módosítása közben
